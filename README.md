# Les mots de l'extreme droite sur YouTube — Analyse textométrique

## Description
Ce projet est une analyse textométrique des discours produits par des
youtubeurs de la Fachosphère française, réalisée dans le cadre d'un
Master à l'Université de Lille. À partir d'un corpus de 28 vidéos issues
de 13 chaînes YouTube postées entre 2019 et 2022, l'objectif est de dresser
une typologie des profils et des stratégies lexicales de l'extrême droite
sur YouTube.

## Compétences mobilisées
- Constitution et nettoyage d'un corpus textuel issu de sources numériques
- Analyse textométrique : fréquences lexicales, cooccurrences, structures
de discours
- Lemmatisation et étiquetage morphosyntaxique avec TreeTagger
- Visualisation des résultats : histogrammes comparatifs par catégorie
grammaticale et par locuteur
- Programmation Python : traitement de corpus, export CSV et Excel,
visualisation avec matplotlib et seaborn

## Contenu
- Construction du tableau lexical global du corpus
- Analyse des 20 mots les plus fréquents par youtubeur
- Comparaison linguistique approfondie entre deux profils (Le Raptor vs
Barbare Civilisé) : noms, verbes, pronoms, adverbes
- Rapport d'analyse complet disponible en PDF
